# Spring Security JWT

## 기술 스택
* IDE
```
 - IntelliJ IDEA
```
* Language
```
 - Java
```
* Back-end
```
 - Spring Boot
 - Spring Security
 - Maven
 - Lombok
 - Java JWT
```
* Database
```
 - MySQL
```
</br>

## 프로젝트 목적
* JWT의 사용 이유와 JWT를 이용한 클라이언트 인증 과정을 파악한다.

* JWT를 이용한 회원가입, 로그인 서버의 구현 방법을 터득한다.
</br>


## 프로젝트 기능
* 회원가입 기능

* JWT를 이용한 로그인 인증 기능

* JWT를 이용한 권한 페이지 접근 기능
</br>
